Sobre mim.

Desde criança, tive contato com a tecnologia e meu interesse por desenvolvimento de software só cresceu desde então. Meu objetivo é trabalhar como desenvolvedor front-end e back-end, bem como analista de dados, em uma empresa que valorize a inovação e a excelência em tecnologia.

Tenho experiência em linguagem de marcação como HTML e CSS e em linguagens de programação como JavaScript, Python, ReactJS e conhecimento em bancos de dados. Além disso, estou comprometido em aprender novas tecnologias e ferramentas para garantir que posso fornecer soluções inovadoras e de alta qualidade.

Estou determinado a trabalhar em equipe para desenvolver soluções de software robustas e escaláveis, fornecer excelente comunicação e fornecer suporte técnico de alto nível para as necessidades da empresa. Com minha experiência e paixão pela tecnologia, estou ansioso para contribuir para o sucesso de sua empresa.

Experiências

Aqui vai algumas experiências que possuo!

Desenvolvimento Web

Decidi aprender mais sobre HTML, CSS e JavaScript para criar meu próprio site do zero. Comecei com HTML, que é a base da estrutura do site. Foi fascinante criar a estrutura das páginas, definir títulos, parágrafos e imagens. Logo depois, mergulhei no CSS, que me permitiu dar estilo ao site. Poder escolher cores, fontes e posicionar os elementos na página me fez perceber como o design é importante.

Provisionamento de REDE

Minha função envolveu configurar e preparar infraestruturas de rede para garantir que sistemas e serviços funcionassem de maneira eficiente. Aprendi a projetar topologias de rede, definir endereçamentos IP e configurar equipamentos como roteadores e switches. No geral, essa experiência me mostrou como o provisionamento de redes é uma mistura empolgante de conhecimento técnico, solução de problemas em tempo real e colaboração com colegas de equipe.

Configuração de ONT, ONU e Roteadores

Em minha trajetória profissional, tive a oportunidade empolgante de trabalhar com a configuração de equipamentos essenciais para redes de comunicação, como ONTs (Optical Network Terminals), ONUs (Optical Network Units) e roteadores. Esse papel envolveu a configuração detalhada desses dispositivos para garantir a conectividade eficiente e confiável dos serviços de rede.

Soluções em Python

Minha jornada profissional incluiu um mergulho empolgante no mundo das soluções em Python. Ao utilizar essa linguagem de programação versátil, fui capaz de criar soluções eficazes para uma variedade de desafios como automação de tarefas repetitivas, Python provou ser uma ferramenta poderosa.Através do desenvolvimento de scripts e programas, pude agilizar processos que antes consumiam tempo e recursos significativos. A clareza da sintaxe de Python tornou a criação de código legível e compreensível, facilitando a colaboração em equipe e a manutenção contínua.

Levantamento de Dados

Minha trajetória profissional me levou ao dinâmico setor de suporte técnico de uma empresa de telecomunicações, onde o levantamento de dados desempenhou um papel crucial. Minha responsabilidade envolveu coletar, analisar e interpretar uma variedade de dados relacionados aos serviços e sistemas de comunicação.

Monitoramento remoto de redes

Minha trajetória profissional me conduziu ao empolgante mundo do monitoramento remoto de redes. Nesse papel, fui responsável por garantir o funcionamento contínuo e a saúde das infraestruturas de rede, mesmo à distância. Utilizando ferramentas avançadas, monitorava parâmetros vitais, como tráfego, latência e disponibilidade. Isso permitia que eu tomasse medidas proativas para resolver problemas antes que afetassem os usuários finais. Ao identificar tendências de desempenho ao longo do tempo, pude realizar ajustes preventivos para otimizar a eficiência da rede.
